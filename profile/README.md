# 🍎 PomoCal

> **"기록하는 시간조차 아까운 사람들을 위해 만들었습니다."**

PomoCal은 뽀모도로 타이머와 캘린더를 하나로 합친 macOS 전용 생산성 도구입니다.  
단순히 시간을 재는 것을 넘어, 당신이 성장에 쏟은 시간을 별도의 노력 없이도 온전한 데이터로 남겨줍니다.

[**Landing Page 바로가기**](https://pomocal.github.io/) | [**Latest Release 다운로드**](https://github.com/PomoCal/pomocal_core/releases)

### 💡 Why PomoCal?

시중에 수많은 타이머 앱이 있지만, '내가 오늘 무엇을 얼마나 했는지'를 확인하려면 결국 다시 손으로 기록해야 했습니다. 그 번거로움이 싫어서, 그리고 매달 나가는 구독료가 아까워서 직접 만들었습니다.

- **기록의 자동화**: 타이머가 끝나면 iCloud 캘린더에 학습 내용이 즉시 기록됩니다.
- **맥락 있는 집중**: Naver Books API를 통해 내가 지금 읽고 있는 책과 학습 세션을 연결합니다.
- **데이터의 주권**: 서버를 거치지 않습니다. 모든 데이터는 오직 당신의 iCloud Drive에만 안전하게 보관됩니다.
- **완전 무료**: 대학생도, 취준생도 부담 없이 사용할 수 있도록 모든 기능은 평생 무료로 제공됩니다.

### 🛠 Tech Stack

macOS 시스템 자원을 최소한으로 사용하면서도 가장 매끄러운 경험을 주기 위해 Native 환경에서 개발하고 있습니다.

- **Language**: Swift
- **Framework**: SwiftUI (macOS 13.0+)
- **Storage**: iCloud Drive (CloudKit)
- **API**: Naver Books Open API
